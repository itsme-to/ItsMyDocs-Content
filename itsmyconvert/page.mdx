## Installation {{ anchor: false }}

To install ItsMyConvert, you need to follow these steps:
1. Download the latest version of ItsMyConvert from the [website](https://builtbybit.com/resources/itsmyconvert.46438).
2. Create a folder on your computer.
3. Put the downloaded file in the folder.
4. Open a terminal in the folder.
5. Run the command `java -jar ItsMyConvert.jar` to launch the CLI.
