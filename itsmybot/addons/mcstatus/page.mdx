# McStatus Addon
MCStatus is an ItsMyBot addon that adds placeholders to retrieve information from a Minecraft server anywhere. ItsMyBot placeholders can be used in any message, other addons, scripts, etc.

## Placeholders
<Properties>
  <Property name="%mcstatus_<server>_players%">
    The amount of players online in the server.
  </Property>
  <Property name="%mcstatus_<server>_max_players%">
    The maximum amount of players the server can have.
  </Property>
  <Property name="%mcstatus_<server>_motd%">
    The message of the day of the server.
  </Property>
  <Property name="%mcstatus_<server>_version%">
    The version of the server.
  </Property>
  <Property name="%mcstatus_<server>_host%">
    The host of the server.
  </Property>
  <Property name="%mcstatus_<server>_port%">
    The port of the server.
  </Property>
  <Property name="%mcstatus_<server>_status%">
    The status of the server.
  </Property>
</Properties>
