# Giveaways Plugins
The giveaways plugins are designed to help you create and manage giveaways in your server. You can create conditional giveaways, set the number of winners, and more.

## Commands
<Properties>
  <Property name="/giveaway create <prize> <time> [winners] [channel] [condition]">
    Create a giveaway.
  </Property>
  <Property name="/giveaway end <message-id>">
    End a giveaway.
  </Property>
  <Property name="/giveaway reroll <message-id> [winners]">
    Reroll a giveaway.
  </Property>
</Properties>
