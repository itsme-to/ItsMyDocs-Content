# ItsMyMeta

ItsMyMeta is a versatile Minecraft plugin that empowers server administrators by providing an easy way to store, manage, and manipulate various types of data. These can range from strings and integers to doubles. ItsMyMeta stands out with its ability to link data with specific team members and tie it to in-game statistics. This offers a comprehensive, adaptable, and efficient solution for managing player data.

# Features

- Easily store and manage strings, integers, and doubles
- Assign data to individual players or entire teams
- Link data to in-game statistics
